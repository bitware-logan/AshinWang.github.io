macOS使用技巧

发表于 2019-05-31 分类于 macOS

记录一些 Mac 的快捷方式，以及好用的软件。

不得不说，在MacBook上，使用触控板和快捷方式，有时候比起鼠标还要方便舒适。

一、快捷使用

1 快速截屏

command+shift+3 截全屏
command+shift+4 选择区域截屏

2 窗口全屏切换

command + option + F窗口全屏切换

comand + shift + F 标签显示切换

3 Chrome强制刷新

command+shift+R
写完博客经常会刷新页面，但是浏览器会有Cookie，于是可采用强制刷新

4 连续看图

像 Window 那样能够左右连续翻看图片，需要选中图片后，按空格键打开，再用上下键切换。

5 快速锁屏

control + command + Q

6 快速桌面

  • 四指上滑，再点击桌面。

  • 重复使用 command + m 隐藏所有应用

7 终端

command + N 新建一个独立终端窗口

command + T新建一个终端窗口

command + shift + T显示命令run详情

8 缩小窗口

command + m

9 隐藏窗口

command + h

command + option + h 隐藏所有窗口

10 切换窗口

control + 方向键

11 关闭 Finder 右侧预览

command + shift + p

二、实用工具

1 硬盘挂载

Mounty-NTFS 工具

  • 如果硬盘为NTFS，可采用Mounty挂载为读写模式。

2 解压归档

The Unarchiver

3 SSH工具

iTerm Putty

4 实时网速

Network Speed Monitor 网速插件

  • 这个插件能够嵌入状态栏和 Show 在 MainWindow,字体大小样式都可自定义设置。

5 清理垃圾

Tencent Lemon Lite 哈哈，免费、无广告，还有网速插件。

Tencent Lemon 更好使，腾讯定义 Mac 版本的腾讯电脑管家！

清理垃圾、测试网速、显示网速（这个最喜欢）。

6 思维导图

XMind

7 MarkDown编辑器

MWeb

MacDown

Typora(推荐)

8 防止自动锁屏和睡眠

Amphetamine

9 压缩视频

HandBrake

10 Win 远程访问

11 LaTex 排版

VSCode

TexPad

12 系统安装启动盘

  • balenaEtcher

13 外接显示器调节

  • MonitorControl

    https://github.com/MonitorControl/MonitorControl

三、其他技巧

1 设置默认打开方式

选中需要的打开的文件，右键 -> 显示简介 -> 打开方式 -> 全部更改

2 隐藏文件

使用chflags hidden <file or directory>隐藏
使用chflags nohidden <file or directory>显示

在 Finder 中，使用Ctrl + shift + .查看或关闭隐藏文件

3 转PDF

选择打印另存 PDF，这个类似 Win10 中的 Microsoft Print to PDF 虚拟打印机

4 外接显示器

  • 假如扩展的显示器在MacBook上面，那么Dock栏要向上转移，在红框鼠标处向下轻扫即可，而不是连接的那部分。
