macOS使用Conda

发表于 2020-02-05 更新于 2020-09-21 分类于 macOS

刷新环境变量：source ~/.bash_profile

进入 (base) 环境

conda命令

查看环境

conda info -e

conda env list

创建环境

conda create --name [虚拟环境名] [python的版本] [需要的包]

激活环境

  • 激活环境

    conda activate [虚拟环境名]

  • 取消激活

    conda deactivate [虚拟环境名]

导出环境

  1. 导出环境的配置 > environment.yaml

conda env export > environment.yaml

  1. 导出 conda list 中的包

conda list -e > requirements.txt

恢复环境

  1. 从 environment.yaml 配置环境中恢复

conda env create -f environment.yaml

  1. 从 requirements.txt 恢复包

conda install --yes --file requirements.txt
