解决kali中文乱码

发表于 2019-12-02 更新于 2020-09-21 分类于 Kali

kali 中文乱码，解决步骤如下。

1 打开list，更改config

vi /etc/apt/sources.list

2 选择原地址

#表示注释，这里我选用阿里云的源

3 刷新源

sudo apt-get update

4 安装中文字体

sudo apt-get install ttf-wqy-zenhei

5 更改本地config

sudo dpkg-reconfigure locales进入配置选项

空格选中/取消，Enter确认

  • en_US.UTF-8
  • zh_CN.GBK
  • zh_CN.UTF-8

6.重启

reboot
