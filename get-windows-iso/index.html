获取Windows-ISO镜像

发表于 2019-12-03 分类于 Windows

在非 Windows 平台，进入 MS 官网即可直接获取。

在 Windows 平台，可通过 Chrome 模拟移动端 OS 获取完整镜像。

使用 Chrome浏览器，F12 打开调试模式。

在调试窗左上角，通过第二个按钮进行模式切换，选择一个手机型号，然后就可以下载啦

  • https://www.microsoft.com/zh-cn/software-download/windows10ISO
  • 选择好版本后会生成临时下载链接，可以使用迅雷等加速下载
