Windows使用技巧

发表于 2019-06-29 分类于 Windows

Windows 平台下的一些快捷方式，和一些实用工具。

一、快捷方式

1 远程桌面

运行 mstsc
运维管理经常用到，不过很多都用堡垒机进去。

2 画图工具

运行 mspaint
低版本Win截图好使，或则贴在word里面。

3 控制面板

运行 control
Win10控制面板不好在开始界面找到了，最初的win10版本右下角Win就有啊「沮丧」。

4 Chrome强制刷新

F5+Ctrl

5 屏幕全键盘

运行 OSK 或 Win + Ctrl + O（win10）
用Excel中，发现方向键无法切换单元格，一些快捷方式也无法使用。上网搜索才知是Scrollock键锁定，公司的键盘不是全键盘，没有此按键，想到可以调用屏幕键盘。

二、坑

1 小键盘切换

Fn＋num lock

键盘突然右部分变成小键盘，例如UIO变成456，搜索才知是数字键锁定了，Fn＋num lock切换解锁。

2 Scrollock键锁

用Excel中，发现方向键无法切换单元格，一些快捷方式也无法使用。上网搜索才知是Scrollock键锁定，公司的键盘不是全键盘，没有此按键，想到可以调用屏幕键盘。

3 键盘冲突

一次，在用Excel进行数组填充时（Ctrl+shift+Enter）,死活不行，，，后来发现是键盘产生多键冲突导致！！！别买便宜国产机械键盘，还不如薄膜好用。。。
