macOS使用pip

发表于 2020-02-19 分类于 Python

pip 使用记录。

安装 pip

  • 脚本安装

  • yum 安装

查看版本

pip -V

pip换源

1 创建配置文件

2 添加镜源配置

vim ~/.pip/pip.conf

3 临时切换源

pip install [package.name] -i [源地址]

安装包

pip install [package.name]

从 txt 需求文件中安装

pip install -r requirements.txt

卸载包

pip unistall [package.name]

查看包

pip list

pip freeze

导出包

将当前环境下的包名和版本导出到 txt 文件

pip freeze -r requirements.txt

更新包

pip install [package.name]

查找包

pip search [package.name]

pip list | grep [package.name]
