Win10添加文件到开始屏幕

发表于 2019-12-02 分类于 Windows
桌面强迫症

日常使用 Windows 中，我会把桌面作为临时工作区。完成工作后，在桌面的文件会归类整理到其他位置，以保持桌面的整洁（不能挡我来来回回换了两小时的壁纸 ：）

  • macOS 10.14内置的Desert 6

所以，快捷方式我一般会放在左下角的开始位置。

但是，最近经常会使用到一个 CHM 的文档，我想把它固定在开始屏幕。

2019/12/12：直接创建快捷方式拖进去进行了，之前弄得太麻烦-_-

1 辅助脚本

新建一个 bat 批处理脚本，命名格式为name.bat让它能打开文档

2 创建快捷方式

将 xxx.bat 文件创建一个快捷方式，再将快捷方式修改好喜欢的 icon 和名字

3 拷贝快捷方式

拷贝快捷方式到系统盘的开始文件夹，随意右键点击一个程序转到文件所在位置，即可到达

4 固定到开始屏幕

在程序列表中找到——>移动至开始屏幕的空白处
